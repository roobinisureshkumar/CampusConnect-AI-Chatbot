# CampusConnect – AI Powered College Chatbot

## 📌 Project Overview

CampusConnect is an AI-powered chatbot designed to provide quick and accurate information about college services.
The system answers student queries related to admissions, courses, examinations, departments, and campus facilities using institutional documents.

## 🎯 Objectives

* Provide instant answers to student queries
* Reduce administrative workload
* Retrieve information from official college documents
* Improve accessibility of campus information

## ⚙️ Technologies Used

* Frontend: React
* Backend: FastAPI
* AI Framework: LangChain
* Vector Database: ChromaDB
* Containerization: Docker

## ✨ Features

* Natural language question answering
* Retrieval-Augmented Generation (RAG)
* Document-based knowledge system
* User-friendly web interface
* Fast response using AI models
